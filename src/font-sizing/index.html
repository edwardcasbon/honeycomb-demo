Font sizing

Honeycomb has defined set font sizes to use on text elements. The sizes are:

  • 36px
  • 24px
  • 18px
  • 16px
  • 14px
  • 12px
  • 10px

Applying colours in CSS

The web toolkit includes a font-size mixin for setting font sizes. The mixin outputs the font size in both px and rem values, while also setting the line height.

The line height can be passed into the mixin, or if omitted will calculate the line height at font-size + 10px, so a 24px font size would result in a 34px line height.

Examples

  • @include font-size(24px); would set a font size of 24px with 34px line height.
  • @include font-size(24px, 48px); would set a font-size of 24px with 48px line height.
