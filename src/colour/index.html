Colour

Please refer to the Honeycomb docs for the latest colour palette.

Applying colours in HTML

Each colour has a HTML class that can be used to apply that colour as a font colour, or background color.

  • color--blue--4 will apply a text colour of #67a9f1 to the element it's applied to.
  • background-color--green--2 will apply a background colour of #a3dea5 to the element it's applied to.
  • color--red will apply a text colour of #cc0000 to the element it's applied to.

Applying colours in CSS

The web toolkit includes a function for getting a colour from the palette. (Note, if the 2nd parameter is omitted it'll default to the 'base' shade)

  • color: color('red', 3); will apply a text colour of #e06666.
  • color: color('red'); will apply a text colour of #cc0000.
