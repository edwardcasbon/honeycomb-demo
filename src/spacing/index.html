Spacing

Honeycomb has a preset scale of values that should be used to apply spacing between elements. These are:

  • 80px
  • 70px
  • 60px
  • 48px
  • 40px
  • 32px
  • 24px
  • 20px
  • 10px
  • 4px

Applying spacing using HTML

The web toolkit provides 3 classes for applying spacing to elements. These are:

  • spaced will apply a margin of 40px.
  • spaced--tight will apply a margin of 20px.
  • spaced--loose will apply a margin of 80px.

To only set spacing on a particular side you can use:

  • spaced-left will apply a left margin of 40px.
  • spaced-left--tight will apply a left margin of 20px.
  • spaced-right--loose will apply a right margin of 80px.

Instead of margin, you can add padding using the padded class name instead of spaced.

Applying spacing in CSS

The web toolkit includes mixins for applying margin and padding to elements. These are the margin and padding mixins. (Original hey :p)

They work by providing a ratio and an index to the mixin. The ratio is then checked against a whitelist of allowed values, before being used to calculate the margin/padding value.

The whilelists for padding and margin can be overridden in your project by resetting the $hc-margin-whitelist and $hc-padding-whitelist variables.

The value is calculated by multiplying the ratio with the $hc-spacing-unit (currently set at 40px).

Examples

  • @include margin(1, 'left'); will apply a left margin of 40px.
  • @include margin(0.6); will apply a margin of 24px to every side of the element.
